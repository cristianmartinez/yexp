# Getting Started

Get up and running with Jext in minutes.

## Installation

Install Jext via npm or bun:

```bash
npm install @jext/core
# or
bun add @jext/core
```

## Basic Usage

```typescript
import { run } from '@jext/core'

// Prepare your input data
const input = {
  products: [
    { name: 'Laptop', price: 999 },
    { name: 'Mouse', price: 25 },
    { name: 'Keyboard', price: 75 }
  ]
}

// Evaluate an expression (use $ to access input)
const result = run('$.products.filter(p => p.price < 100)', input)
console.log(result)
// Output: [{ name: 'Mouse', price: 25 }, { name: 'Keyboard', price: 75 }]
```

## Context Access

Jext provides three ways to access data in expressions:

- **`$`** - Access the main input data
- **`$context`** - Access auxiliary context (optional)
- **`$env`** - Access environment variables (optional)

```typescript
const input = { items: [1, 2, 3] }
const options = {
  context: { userId: 'usr_123' },
  env: { API_URL: 'https://api.example.com' }
}

run('$.items.length', input, options)  // 3
```

## Compilation & Execution

For better performance, compile expressions once and reuse:

```typescript
import { tokenize, parse, compile, evaluate } from '@jext/core'

// Compile to bytecode
const program = compile(parse(tokenize('$.items |> filter(x => x > 2)')))

// Execute multiple times (fast!)
const result1 = evaluate(program, input1)
const result2 = evaluate(program, input2)
```
