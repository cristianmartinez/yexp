## What is Yexp?

Yexp is an expression engine designed for:
- **Safe evaluation** - No arbitrary code execution
- **Deterministic results** - Same input always produces same output
- **High performance** - Compiles to bytecode for fast execution
- **Familiar syntax** - JavaScript-like with functional programming features

## Quick Example

```javascript
// Filter and transform data
$.products
  .filter(p => p.price < 100)
  .map(p => p.name)
  .join(", ")
```

## Key Features

- ✓ Property access & optional chaining
- ✓ Array methods (filter, map, reduce, etc.)
- ✓ Higher-order functions & lambdas
- ✓ Template literals
- ✓ Pipe operator for composition

- ✓ Spread operators
