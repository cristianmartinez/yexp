# Changelog

All notable changes to Yexp will be documented in this file. The project follows [Semantic Versioning](https://semver.org/) for published packages while the language specification and bytecode maintain their own explicit versions.

## [Unreleased]

### Changed

- Prepared the repository, documentation, package metadata, CI, and release process for public contributions.

## [0.0.1] - 2026-07-11

### Added

- Initial `@cristianmartinez/yexp` expression-language package.
- Initial `@cristianmartinez/yexp-cli` terminal package with the `yexp` binary.
- Lexer, parser, bytecode compiler, VM, built-in functions, and JSON/NDJSON CLI workflows.
- Interactive playground and language documentation.
